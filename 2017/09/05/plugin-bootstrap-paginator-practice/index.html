bootstrap-paginator插件实践

发表于 2017-09-05 | 分类于 前端

bootstrap-paginator是bootstrap的分页插件。

此文进行使用实践。

依赖引入

javascript：

  • jquery.js
  • bootstrap-paginator.js

css：

  • bootstrap.css

分页思路

  • 后端分页：前端显示第一页的10条数据，则请求后端去数据库查询符合条件的前10条；若需要显示第二页的10条数据，则请求后端去数据库查询符合条件的第11-20的10条数据。
  • 前端分页：一次性请求完所有符合条件的数据，保存在数组中，前端的每次分页都从这个数组里面去取出。

本文的分页基于后端分页。
