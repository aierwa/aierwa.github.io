选择框脚本（select）

<select>具有如下属性方法：

  • add(newOption, relOption)：插入
  • multiple：是否多选
  • options：所以option的集合
  • remove(index)：移除
  • selectedIndex：基于0的选中项的索引，如果没有则为-1
  • size：选择框的可见行数
  • type：根据multiple的值而为”select-one”或”select-multiple”
  • value：没有选中项为””，有选中项为选中项的value值，没有value值就为起文本值。

<option>具有如下属性：

  • index：选项在options集合中的索引
  • lable：选项的标签
  • selected：是否选中
  • text：其文本
  • value：其value值
