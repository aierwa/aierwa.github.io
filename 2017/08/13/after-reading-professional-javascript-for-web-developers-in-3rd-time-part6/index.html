HTML5 脚本编程

跨文档消息传送

cross-document messaging，简称XDM。

核心是postMessage()，用于传递参数到当前页面中的<iframe>元素，或者由当前页面弹出的窗口。
两个参数：消息（最好是字符串）；消息接收方来自哪个域。

若传其他，可考虑JSON.stringify(), JSON.parse()。
IE8+支持。目前XDM已单独规范了，叫Web Messaging。

接收到XDM消息时，会触发window对象的message事件。
